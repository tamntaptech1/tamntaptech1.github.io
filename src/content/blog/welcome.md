This blog is built with modern web technologies including:

- **Astro 5** - A modern web framework for building fast, content-focused websites
- **TypeScript** - For type-safe development
- **Tailwind CSS** - For utility-first styling
- **GitHub Pages** - For hosting and deployment

## Features of This Blog

This blog includes several features to enhance your reading experience:

### Dark Mode

Toggle between light and dark themes using the button in the navigation bar. Your preference is saved and persists across sessions.

### Search

Use the search feature to find articles by keywords, topics, or phrases. The search is powered by Pagefind and works offline.

### Table of Contents

Longer articles include a table of contents on the right side for easy navigation.

### RSS Feed

Subscribe to the RSS feed to get notified when new articles are published.
